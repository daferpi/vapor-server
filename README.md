# vapor-server
Swift Vapor Server example, how to work with swift in server side

vapor new project
-----------------
vapor new "Project name"

convert project to xcode
------------------------
vapor xcode 

build project
-------------
vapor build

run project
------------
vapor run

call to show json
-----------------
http://localhost:8080/hello

------------------
